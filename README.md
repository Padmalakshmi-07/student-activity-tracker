#  Student Activity Tracker Web Application

A simple and interactive front-end web application designed to help students track their daily learning activities and monitor their progress through a clean and user-friendly interface.

---

##  Project Overview

This project is developed as part of the **Naan Mudhalvan Hackathon – Level 2 (Interactivity Implementation)**. The main objective is to convert a static user interface into an interactive application using JavaScript.

The application allows users to update activity status dynamically and view real-time progress without reloading the page. It demonstrates core front-end development concepts such as event handling, DOM manipulation, and dynamic UI updates.

---

##  Features

-  **Home Section**
  - Displays welcome message and motivational quotes  
  - Includes “Start Tracking Activities” button for quick navigation  

-  **Activities Section**
  - Displays list of tasks dynamically  
  - Shows task status (Pending / Completed)  
  - Allows users to mark tasks as completed  

-  **Progress Section**
  - Displays number of completed tasks  
  - Shows progress using an animated and glowing progress bar  

-  **Real-Time Updates**
  - Updates UI instantly without page reload  
  - Dynamically updates task status and progress  

-  **Navigation System**
  - Smooth switching between Home, Activities, and Progress sections  

---

##  Technologies Used

- **HTML** – Structure of the application  
- **CSS** – Styling, layout, and visual effects  
- **JavaScript** – Interactivity, event handling, and dynamic updates  

---

##  How It Works

- Activities are stored in a JavaScript array  
- Each activity contains a name and status (Pending / Completed)  
- When the user clicks “Mark Completed”:
  - The status updates from Pending to Completed  
  - The UI updates instantly using DOM manipulation  
  - The progress count and progress bar are updated automatically  

---

##  How to Run the Project

1. Download or clone the repository  
2. Open the project folder  
3. Double-click on `index.html`  
4. The application will open in your browser
